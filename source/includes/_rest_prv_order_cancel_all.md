### Cancel All Open Orders

> Response

```json
{
    "code": 0
}
```

**HTTP Request**

`DELETE /<grp>/api/pro/v2/futures/order/all`

**Prehash String**

`<timestamp>+v2/futures/order/all`

**Request Parameters**

PARAMETER | TYPE   | REQUIRED | DESCRIPTION
--------- |--------| -------- | ------------------------------------------------------ 
symbol    | String |   No     | the optional symbol filter
